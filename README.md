# Tic Tac Toe client by React / Redux

Backend server at https://github.com/shuvro/mern-tic-tac-toe-server.git
 
### Start

````bash
yarn
````

````bash
yarn build:dev
````

````bash
yarn serve:dev
````

````bash
http://localhost:3000/
````
### Docker
Run the Game Frontend (React/Redux) using docker

```bash
docker-compose up --build
```

Checkout the application at 
```bash
http://localhost:3000
```
